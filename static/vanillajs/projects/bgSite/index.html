i've decided websites all look the same... kind of like magazine ads. just google 'best website designs' and you'll see what i mean. so i've taken on the task of making a few proof-of-concepts. i'm generally aiming for fun, animated, or interactive UIs. whatever that means. this particular one has p5.js running in the background using the 3d 'sine-cosine' demo on their website, slightly modified. on top i've put a simple collapsible menu, so you can always collapse it and watch the show. the collapsible elements can be any HTML you want, as big as you want. I'm just using bootstrap components for simplicity. you could also modify the sidebar to have it follow as you scroll down, or have a tab open when you load the page (like I do with 'donate' -- spot the differerence in the code to see how), or have your github load into a collapsible panel instead of redirecting your window... whatever!
